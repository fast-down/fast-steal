`fast-steal` 是一个特别快的多线程库，支持超细颗粒度的任务窃取。

## 优势

1. 零依赖
2. 无锁
3. 零拷贝
4. 安全的 Rust 代码
5. 除 `thread` 外，实现 `no_std`
6. 超细颗粒度任务窃取，速度非常快
